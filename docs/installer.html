Installer Sésame

Suivez ces 4 étapes pour installer l'application sur votre téléphone Android.

⬇ Télécharger Sésame

Étape 1 — Télécharger le fichier

1
Appuyez sur le bouton bleu ci-dessus. Le fichier sesame.apk se télécharge dans votre dossier Téléchargements.

Étape 2 — Autoriser l'installation

2
Android affiche un avertissement la première fois. C'est normal pour une application hors Play Store.

Appuyez sur Paramètres dans le message, puis activez « Autoriser depuis cette source ».
💡 Sur certains téléphones, ce réglage se trouve dans Paramètres → Applications → Autoriser les sources inconnues ou Paramètres → Sécurité.

Étape 3 — Installer

3
Revenez sur le fichier téléchargé (dans les notifications ou dans le dossier Téléchargements) et appuyez sur Installer.

Étape 4 — Ouvrir Sésame

4
L'application apparaît sur votre écran d'accueil. Ouvrez-la et sélectionnez les raccourcis qui vous intéressent.
✅ Vos identifiants restent sur votre téléphone. Sésame ne transmet aucune donnée à internet.
